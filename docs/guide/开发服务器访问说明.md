# 开发服务器访问说明

## 后台管理开发服务器

### 服务器信息

- **服务器IP**：`23.248.226.82`
- **开发端口**：`1820`（默认，可在配置文件中修改）
- **后台管理路径**：`/admin`

### 完整访问URL

#### 方式1：通过服务器IP访问
```
http://23.248.226.82:1820/admin
```

#### 方式2：通过本地访问（如果在本机）
```
http://localhost:1820/admin
```

#### 方式3：通过127.0.0.1访问
```
http://127.0.0.1:1820/admin
```

### 启动命令

```bash
cd /www/wwwroot/23.248.226.82/web
pnpm dev
```

或指定端口：
```bash
pnpm dev --port 5657
```

### 配置文件

**文件位置**：`web/.env.development`

```env
# 本地环境
ENV = 'development'

# base路径
VITE_BASE_PATH = './'

# 本地环境接口地址 - 尾部无需带'/'
VITE_AXIOS_BASE_URL = http://47.76.239.170:8080

# 开发服务器端口
VITE_PORT = 1820
```

### 注意事项

1. **后台管理入口**：访问 `/admin` 路径会自动重定向到 `/admin/loading`
2. **API接口地址**：前端会连接到 `http://47.76.239.170:8080`（后端API服务器）
3. **热更新**：开发模式下支持热更新，修改代码后自动刷新
4. **跨域**：如果遇到跨域问题，检查后端API的CORS配置

### 停止服务器

```bash
# 查找进程
ps aux | grep vite

# 停止进程（替换PID为实际进程ID）
kill <PID>

# 或使用端口查找并停止
lsof -ti:5657 | xargs kill
```

### 检查服务器状态

```bash
# 检查端口是否监听
netstat -tlnp | grep 5657
# 或
ss -tlnp | grep 5657

# 测试访问
curl http://localhost:5657
```
