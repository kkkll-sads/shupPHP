# API 接口变更说明 (2025-12-28)

## 1. 藏品详情 (Detail)

**接口地址**: `/api/collectionItem/detail`
**请求方式**: `GET`

### 变更内容
- **移除字段**: `core_enterprise`, `farmer_info`
- **新增/确保字段**: `price_zone` (价格分区ID)

### 响应示例
```json
{
    "code": 1,
    "msg": "",
    "data": {
        "id": 27,
        "title": "华东数链权益",
        "price": "449.94",
        "price_zone": 1,           // 新增/确保返回
        "image": "...",
        "stock": 0,
        "status": "1",
        "sales": 5,
        "content": "...",
        "asset_code": "37-DATA-0007-0001",
        "fingerprint": "0x0f419a32...",
        // "core_enterprise": "...", // 已移除
        // "farmer_info": "..."      // 已移除
    }
}
```

## 2. 原始藏品详情 (Original Detail)

**接口地址**: `/api/collectionItem/originalDetail`
**请求方式**: `GET`

### 变更内容
- **移除字段**: `core_enterprise`, `farmer_info`
- **新增/确保字段**: `price_zone` (价格分区ID)

## 3. 专场藏品列表 (By Session)

**接口地址**: `/api/collectionItem/bySession`
**请求方式**: `GET`

### 变更内容
- **逻辑变更**: 将同一资产包（`package_name` + `zone_id`）下的所有商品（官方库存 + 寄售）聚合为一个条目返回。不再分别返回官方商品和寄售商品。

### 响应数据结构更新
```json
{
    "code": 1,
    "msg": "",
    "data": [
        {
            "id": 26,                  // 聚合后的代表ID（通常是第一个找到的商品ID）
            "title": "华东数链权益",
            "package_name": "华东数链权益",
            "zone_id": 1,
            "session_id": 3,
            "image": "...",
            "price": "419.60",         // 代表价格（兼容旧版）
            "stock": 15,               // 代表库存（兼容旧版，官方库存）
            
            // 新增聚合统计字段
            "official_stock": 15,      // 官方剩余库存
            "consignment_count": 3,    // 寄售单数量
            "total_available": 18,     // 总可用数量 (official_stock + consignment_count)
            
            "min_price": 399.62,       // 最低价（包含官方和寄售）
            "max_price": 501.80,       // 最高价
            "official_min_price": 419.60, // 官方最低价
            "consignment_min_price": 399.62, // 寄售最低价
            "price_range": "399.62 - 501.80", // 价格范围字符串
            
            // 寄售商品列表（新字段）
            "consignment_list": [
                {
                    "id": 147,
                    "price": "399.62",
                    "avatar": "...",   // 卖家头像
                    "nickname": "..."  // 卖家昵称
                },
                // ...
            ],
            
            // 兼容性字段
            "is_consignment": 0,       // 聚合项默认为0，具体看 consignment_list
            "consignment_price": 0
        }
    ]
}
```
