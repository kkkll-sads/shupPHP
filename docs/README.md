# 项目文档索引

本文档目录结构如下：

## 1. 架构文档 (`architecture/`)
核心技术架构与设计方案。
- `系统架构文档.md`: 整体系统架构概览
- `财务架构与资金流向分析.md`: 核心资金流设计
- `数据库架构分析_*.md`: 数据库结构设计

## 2. 功能说明 (`features/`)
具体业务模块的功能逻辑说明。
- `权益交割功能说明.md`: 权益交割业务逻辑
- `矿机与分红系统说明.md`: 矿机业务逻辑
- `代理佣金系统说明.md`: 代理佣金计算逻辑
- `用户预约撮合逻辑说明.md`: 盲盒预约与撮合
- `藏品状态保护规则.md`: 藏品流转状态机

## 3. 接口参考 (`reference/`)
API 接口定义与字段说明。
- `API响应字段规范化说明.md`: 统一 API 返回格式规范
- `money字段架构调整说明.md`: 资金字段重构说明
- `字段对照表.md`: 数据库/API 字段映射

## 4. 更新日志 (`changelogs/`)
版本迭代与变更记录。
- `更新日志_*.md`: 每日开发日志
- `API接口变更_*.md`: 接口变更记录

## 5. 维护与报告 (`maintenance/`)
问题修复、数据清理与验证报告。
- `修复_*.md`: 特定 Bug 的修复记录
- `数据库性能优化-寄售表索引.md`: 数据库优化报告
- `数据库索引优化执行报告.md`: 索引执行结果
